## About REO Mobile

**REO Mobile** is a mobile application developed by **SayLess** to streamline and modernize the operations of the **Research Extension Office (REO)**.

Traditionally, research-related processes such as submissions, documentation, and approvals can be slow and paper-based. REO Mobile transforms these workflows into a **faster, digital, and more accessible system**.

The application extends the existing **REO web platform** into a mobile environment, allowing users to conveniently interact with the system using their smartphones.

---

## Purpose of this Repository

This repository exists **primarily for application distribution**.

It serves as a central place where users can:

- Download the official **REO Mobile application**
- Access the latest releases
- Receive updates from the development team

The **source code is not publicly distributed in this repository**.

## Download the App

You can download the latest version of the application from the **Releases** section.

➡ **[Download Latest Version](../../releases)**

### Installation Guide

1. Download the latest `.apk` file from the Releases page.
2. Open the downloaded file on your Android device.
3. If prompted, enable **Install from Unknown Sources**.
4. Complete the installation.

## Key Features

- Mobile access to REO services
- Simplified research submission process
- Faster tracking of research documentation
- User-friendly interface
- Integration with the existing REO web system

---

## Target Users

REO Mobile is designed for:

- Students conducting research
- Faculty researchers
- Research coordinators
- External researchers collaborating with the institution
